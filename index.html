VISUAL STUDIO CODE

Visual Studio Code is a streamlined code editor with support for development operations like debugging, task running, and version control. It aims to provide just the tools a developer needs for a quick code-build-debug cycle and leaves more complex workflows to fuller featured IDEs, such as Visual Studio IDE.

Visual studio features

Code editing: Support for hundreds of languages, including JavaScript, TypeScript, and Node:

Syntax highlighting

Bracket-matching

Auto-indentation

Box-selection

Snippets

Debugging: Set breakpoints, inspect variables, and step through code execution:

Integrated debugging capabilities

Code completion: IntelliSense features like code completion, parameter info, quick info, and member lists

Version control: Embedded version control with Git

Customization: Change the theme, keyboard shortcuts, and preferences

Extensions: Install extensions to add functionality

Command Palette: Access all commands by typing keywords

User interface: Navigate to any file or symbol by typing its name using Ctrl+P

HTML Emmet

Emmet symbol Description
> Used to add a child element to an abbreviation
Plus operator(+) Used to create sibling elements.
Dollar sign($) Used to output the current number of a repeated element.
semicolon(;) Used with the letter w to create width:
Colon(:) Used with the letter h to create height:.
Exclamation point(x) Used as an alias for ex.
Percentage sign(%) Used as an alias for p.
